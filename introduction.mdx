## Setting up

You can self-host OpenCopilot in a relatively easy way, please make sure you have the following requirements:

- Docker engine and docker compose installed
- Clone the repository

```bash
git clone https://github.com/openchatai/opencopilot.git
```

- Update your llm-server/Dockerfile with your OPENAI_API_KEY

```dotenv
KEY OPENAI_API_KEY YOUR_KEY_GOES_HERE
```

- Run the installation script

```bash
make install
```

Then your OpenCopilot dashboard will be accessible at http://localhost:8000

Complete list of ports:

- http://localhost:5000 -> backend APIs (create, update, delete and validate copilots)
- http://localhost:8000 -> dashboard (UI to do the same)
